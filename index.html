About
Straker is a self-contained portfolio tracker. All data is stored locally on this device (localStorage) — nothing is sent to any server except live price/news requests to Yahoo Finance (via your configured proxy). Add this page to your iOS Home Screen (Share → Add to Home Screen) for an app-like experience.
